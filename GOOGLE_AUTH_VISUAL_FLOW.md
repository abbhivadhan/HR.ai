### Key URLs
- Login: `/auth/login`

- Callback: `/auth/callback`
- Dashboard: `/dashboard`

### Key Components
- `AuthContext.tsx` - State management
- `LoginForm.tsx` - Login UI
- `RegisterForm.tsx` - Register UI
- `GoogleButton.tsx` - Reusable button
- `callback/page.tsx` - OAuth handler

### Key Functions
- `loginWithGoogle()` - Initiates OAuth
- `signInWithOAuth()` - Supabase helper
- `handleCallback()` - Processes redirect
- `createProfile()` - Creates user profile

### Key States
- `isLoading` - OAuth in progress
- `isAuthenticated` - User logged in
- `user` - User data object
- `error` - Error message
